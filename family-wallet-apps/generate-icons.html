🎨 Icon Generator — Family Wallet PWA

Klik tombol di bawah untuk generate & download semua icon yang dibutuhkan PWA.

Cara pakai:
1. Klik Download Semua Icon
2. Buat folder icons/ di repo GitHub kamu
3. Pindahkan semua file icon-XXX.png ke dalam folder tersebut
4. Pastikan struktur repo seperti ini:

📁 repo-kamu/
  📄 index.html
  📄 manifest.json
  📄 sw.js
  📁 icons/
    📄 icon-192.png
    📄 icon-512.png
    📄 ... (dan seterusnya)
